- Для простоты тестирования, конфигурации писем берутся из файла json, функцией get_json_confs()
- Возможность запуска до 2 потоков задаётся опцией: "--autoscale=1,2"
- В случае неудачи отправки будет предпринято ещё 2 попытки т.к. retry_kwargs={'max_retries': 2}
- Запуск командой: "celery -A tasks worker --autoscale=1,2 --loglevel=info"
